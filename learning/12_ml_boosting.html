Gradient Boosting & XGBoost

¿Cómo ganar competencias de Kaggle? En lugar de hacer un árbol gigante, entrena 100 árboles diminutos, donde cada uno corrige los errores del anterior.

Área Machine Learning Duración ~15 min Prerrequisitos Árboles de Decisión
Ensamblando Modelos

Ajustando los Residuos

Mientras que Random Forest promedia muchos árboles independientes, el Gradient Boosting es secuencial. Funciona así:

  1. El Árbol 1 hace una predicción burda. Queda un error (Residuo).
  2. El Árbol 2 no intenta predecir el valor original, intenta predecir el Residuo del Árbol 1.
  3. El Árbol 3 predice el residuo que quedó tras el Árbol 2.

Al sumar todas las predicciones de estos "árboles débiles", obtenemos un modelo extremadamente preciso. XGBoost es simplemente una versión ultra-optimizada matemáticamente de esta misma idea.
