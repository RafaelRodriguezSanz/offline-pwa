El Meta-Algoritmo

Computación Evolutiva

A veces, Backpropagation no sirve. Si la función matemática que intentas optimizar tiene quiebres repentinos, no es diferenciable, o las reglas son opacas (ej. el puntaje final de un videojuego), no hay gradiente que calcular.

Los Algoritmos Genéticos (GA) mantienen una población de múltiples soluciones posibles simultáneamente. Cada solución se evalúa, las mejores sobreviven, y se mezclan para crear la siguiente generación. Es un proceso de "prueba y error guiado".

Conceptos Clave

Vocabulario Darwiniano

  • Cromosoma: La representación de una solución. Puede ser un string binario `100110` o un vector de números reales `[1.4, -0.2]`.
  • Fitness Function: La función de aptitud. ¿Qué tan buena es esta solución? Es la "madre naturaleza" que decide quién vive.
  • Selección: El método para elegir padres. Ruleta (probabilidad proporcional al fitness) o Torneo (eliges a 3 al azar y el mejor gana el derecho a reproducirse).
